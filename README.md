# おえかき＆ぬりえ（子ども向けPWA）

iPad mini で子どもが直感的に遊べる、お絵描き＆塗り絵の Web アプリ。
ビルドツール不要の静的な HTML/CSS/JS のみで動きます。

## 機能

- **バケツ塗り**：SVG線画の境界を尊重して塗る（線画をマスクにして色漏れ・白フチを防止）
- **ペン / 消しゴム**：太さをスライダーで調整（縦スライダー）
- **16色パレット**（白を含む）
- **塗り絵の切り替え**：`images/` の SVG から選択（しろ紙＝白紙キャンバスも）
- **保存**：作品をPNGで書き出し（iPad は共有シート、または画像を長押しして「写真に追加」）

## つかいかた（ローカル）

`fetch` を使うため、`file://` 直開きではなくローカルサーバー経由で開きます。

```
# 例：Node がある場合
npx serve .
# もしくは Python
python -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開く。

## 塗り絵を追加する

1. `images/` に SVG を置く（輪郭を閉じた線画にする）
2. `images/list.json` にファイル名を追加する

```json
["fish.svg", "car.svg", "..."]
```

> ローカル開発サーバーによっては `images/list.json` を自動生成できますが、
> GitHub Pages などの静的ホストでは **この `list.json` が一覧の元になる**ため、
> デプロイ前に必ず更新してください。

## 公開

GitHub Pages / Netlify など、アプリと同じオリジンで配信してください
（外部URLからの画像読み込みは Tainted Canvas によりバケツ塗りが機能しなくなります）。
